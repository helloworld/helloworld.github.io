The ultimate planning app for hackathon organizers. Hackers can create a profile with all there info when they sign up for hackerverse. It allows you to view attendees, stats on which school hackers are coming from, how many t-shirts of each size to order, what type of meals to plan, and much more.

ePush

ePush allows one to easily deploy files to the cloud... via email! Just email us with the subject 'set me up' to get a verification code. Then, simply send a message with your verification code in it, with the files (or github repo link) you wish to upload to the same address. Emails sent are detected realtime via webhooks, and your files are then dynamically sandboxed on our servers!

This app won the Pebble Sendgrid prize at HackRU
